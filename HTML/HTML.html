-> HTML is stands for HyperText Markup Language.

-> standard markup language

-> If you want to create a Website then..
   HTML is must needed.

-> HTML is like body, it gives static skeleton to web application and website.




STUCTURE:

<!DOCTYPE html> : Describes the type of document. [This is html document]

<html lang="en"> : Describes language of document. [This document is in English language]

<head> : 1. Not display on webpage
         2. Running in background [CSS, JS]
         3. Display on title [<title> Title Tag </title>]
         4. <meta>, <title></title>, <link> [External Style Sheet]


[ <meta> : - Meta Tag is Describe data about the data
           - Meta Tag is used to Describe.. that keywords, descriptions which is used in your website
           - Meta Tag is mainly used for SEO ]

         SEO : Search Engine Optimization
         (SEO is used to generate web traffic)


         <meta charset="UTF-8"> : - Specifies the character 
                            - UTF-8 covers almost all of the characters and symbols in the world!
                             
         <meta http-equiv="X-UA-Compatible" content="IE=edge"> : - The X-UA-Compatible meta tag allows web authors to choose what version of
                                                             Internet Explorer
                                                           - Edge mode tells Internet Explorer to display content in the highest mode
                                                             available.

         <meta name="viewport" content="width=device-width, initial-scale=1.0"> : To make your website look good on all devices

         <meta name="keywords" content="HTML, CSS, JavaScript"> : Define keywords for search engines

         <meta name="description" content="Free Web tutorials for HTML and CSS"> : Define a description of your web page

         <meta name="author" content="John Doe"> : Define the author of a page

         <meta http-equiv="refresh" content="30"> : Refresh document every 30 seconds

         <title> Document </title> : Specifies title of the document
