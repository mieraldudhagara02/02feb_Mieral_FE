Q-1    Are the HTML Tags and Elements are the same thing?

-   No, HTML Tags and Elements are not the same thing, but we interpret that they are same. Let's see the difference between HTML Tags and HTML Elements...

HTML TAGS:
-   In HTML, Most of Tags are exists in pair. So, we can say that they are Pair Tags.
-   These type of Tags are starts with a < bracket and ends with a > bracket.
-   In Pair Tags, first tag is Opening Tag and second tag is Closing Tag. Both are same, but Closing Tag is ends with /.
-   Some HTML Tags did not have Closing Tag. Thus, these Tags are Unpair Tags.
      Here, <p> is Opening Tag.
                </p> is Closing Tag.
                Thus, this is Pair Tag.
                <link> is Unpair Tag. Because <link> Tag does not have Closing Tag.

HTML ELEMENTS:
-   HTML Element includes a start tag, content, and an end tag.
-   Some Elements did not need to have content inside it. Because some tags are Unpair Tags so.
      Here, <p> <-Opening Tag [This is Some Content.] Closing Tag-> </p>

Q-3    What are Void Elements in HTML?

How!! Let's See..
-   Most of the HTML elements have start and end tags.
