3. SEMANTIC ELEMENT IN HTML5.

INTRODUCTION:

    - A semantic element clearly describes its meaning.
    - Semantic elements have meaningful names which tells about the content.

Let's see one by one with Syntax and Example:

1. <header>
    - <header> tag is used to define the header for a document.
    - <header> tag containes:
        * Heading Elements [<h1> to <h6>]
        * Logo or Icon

SYNTAX:

<header>
      //content
</header>

EXAMPLE:

  • VEHICLES
  • PURCHASE
  • OWNERS
  • EXPLORE

2. <nav>
    - When you don't want to create a header in your website, instead of header you can create navigation.
    - If you want to create a navigation in your website, you need <nav> tag.

SYNTAX:

<nav>
      //content
</nav>

EXAMPLE:

  • VEHICLES
  • PURCHASE
  • OWNERS
  • EXPLORE

3. <section>
    - <section> tag is used to define sections in a document.
    - When you put your content on a web page, it may contains many chapters, headers, footers.
    - That's why we need to use <section> tag.

SYNTAX:

<section>
      //content
</section>
