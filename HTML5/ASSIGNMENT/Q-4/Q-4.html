4. CANVAS AND SVG TAG.

1. SVG Tag:

INTRODUCTION:

    - SVG : Scalable Vector Graphics.
    - SVG defines vector-based graphics.
    - SVG is W3C recommendation.
    - SVG images are Zoomable, which means images do not lose their quality when zoomed.
    - By using SVG, we can draw paths, boxes, circles, text etc.
    - SVG images can be created with any text editor.

SYNTAX 1:

<svg width="value" height="value">
    <circle cx="value" cy="value" r="value" stroke="value" stroke-width="value" fill="value">
</svg>

Here,

- SVG image begins with an <svg> element and ends with </svg>.
- HEIGHT : define the height of svg image.
- WIDTH : define the width of svg image.
- <CIRCLE> : is used to draw a circle.
- cx : define the center of the circle on the x-axis.
- cy : define the center of the circle on the y-axis.
- r : define the radius of the circle.
- STROKE : stroke means border.
- STROKE-WIDTH : define width of the stroke.
- FILL : define color inside the circle(shape).

<svg width="value" height="value">

- rx : rounds the corners of the rectangle.

- STROKE-WIDTH : define width of the stroke.
- FILL : define color inside the circle(shape).

2. CANVAS Tag:

INTRODUCTION:

    - The HTML <canvas> element is used to draw graphics on a web page.
    - We need JavaScript, for drawing graphics by using <canvas> element.

SYNTAX:

<canvas id="value" width="value" height="value">
    //canvas code
</canvas>

EXAMPLE :


NOTE:   You must have to specify id. (for JavaScript)
    - After creating the rectangular canvas, you must add a JavaScript to do the drawing.

DRAW A LINE:

SYNTAX:

<canvas id="value" width="value" height="value"></canvas>
<script>
    var variable_name = document.getElementById("value");
    var context = variable_name.getContext("value");
    variable_name.moveTo(x,y);
    variable_name.lineTo(200,100);
    variable_name.stroke();
</script>

Here,

- id : The <canvas> element must have an id attribute(for JS)
- HEIGHT : define the height of canvas.
- WIDTH : define the width of canvas.
- getElementById() : it's a DOM Method(for find the canvas element)
- getContext() : it's built-in HTML function.
- moveTo() : defines the starting point of the line.
- lineTo() : defines the ending point of the line.
- stroke() : It's a ink method (for drawing a line you must use this method like stroke())
