1. WHAT ARE THE NEW TAGS ADDED IN HTML5?

INTRODUCTION:

-   HTML5 is 'markup language.'
-   HTML5 is used to 'structuring' and 'presenting' content on the "World Wide Web."


NEW TAGS ADDED IN HTML5:

1. <audio>
    - HTML audio tag is used to define sounds such as music and other audio clips.
    - There are three supported audio formats in HTML:
          * mp3
          * wav
          * ogg

SYNTAX:

<audio controls autoplay loop muted>
     <source src="URL" type="audio_formate">
     Your browser does not support the html audio tag.
</audio>

Here,
  controls, autoplay, loop, muted all are the Attributes of <audio> tag.

  CONTROLS : displayed play/pause buttons
  AUTOPLAY : specifies that audio will start as soon as it is ready
  LOOP : specifies that audio file will be start again when it's compeleted
  MUTED : used to mute the audio

2. <video>
    - The HTML video tag is used for streaming video files.
    - There are three supported audio formates in HTML:
          * mp4
          * webM
          * ogg

SYNTAX:

<video controls autoplay loop muted height="value" width="value" poster>
     <source src="URL" type="audio_formate">
     Your browser does not support the html audio tag.
</video>

Here,
  controls, autoplay, loop, muted, height, width, poster all are the Attributes of <audio> tag.

  CONTROLS : displayed play/pause buttons
  AUTOPLAY : specifies that video will start as soon as it is ready
  LOOP : specifies that video file will be start again when it's compeleted
  MUTED : used to mute the video
  HEIGHT : Sets the height of the video
  WIDTH : Sets the width of the video
  POSTER : Specifies an image to be shown while the video is downloading, or until user play the video

16. <time>
    - <time> tag is used to define date and time.
    - It displays time value in a 24 hour clock.
    - It helps search engines to produce smarter search results.

SYNTAX:

<time datetime="value">
      //date and time
</time>

Here,
    Three ways of putting value in side the <time> tag.
        1. <time> 07:00 </time>
        2. <time datetime="2002-08-11"> next wednesday </time>
        3. <time datetime="2002-08-11 T0 7:00-09:00"> 7pm last wednesday </time>

    - We need JavaScript, for drawing graphics by using <canvas> element.

SYNTAX:

<canvas id="value" width="value" height="value">
    //canvas code
</canvas>

<canvas id="value" width="value" height="value"></canvas>
<script>
    var variable_name = document.getElementById("value");
    var context = variable_name.getContext("value");
    variable_name.moveTo(x,y);
    variable_name.lineTo(200,100);
    variable_name.stroke();

- id : The <canvas> element must have an id attribute(for JS)
- HEIGHT : define the height of canvas.
- WIDTH : define the width of canvas.
- getElementById() : it's a DOM Method(for find the canvas element)
- getContext() : it's built-in HTML function.
- moveTo() : defines the starting point of the line.
- lineTo() : defines the ending point of the line.
- stroke() : It's a ink method (for drawing a line you must use this method like stroke())
