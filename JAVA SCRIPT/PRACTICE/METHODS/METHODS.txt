1. BOM Method:
   -----------

-> BOM stands for 'Browser Object Model.'

-> The BOM allows JavaScript to 'interact with' the browser.

        * Windows Method:

            - alert()
            - confirm()
            - prompt()

- alert()
-> Using alert you can pass the pop-up box.
    SYNTAX:
        1. window.alert("message");
        2. alert(message);

- confirm()
-> Using conform you can take the choice from the user.
    SYNTAX:
        1. window.confirm(message);
        2. confirm(message);

- prompt()
-> Using prompt you can take the input from the user.
    SYNTAX:
        1. window.prompt(message);
        2. prompt(message);
