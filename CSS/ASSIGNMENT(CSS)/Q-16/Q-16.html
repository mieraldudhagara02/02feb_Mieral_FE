QUESTION-16

What are the advantages and disadvantages of using external style sheet?




EXTERNAL CSS:

SYNTAX: p{color: pink;}
-   In "external css", we have to Create separate .css file and then attach this file in our HTML document through HTML <link> tag.

ADVANTAGES AND DISADVANTAGES OF EXTERNAL CSS
ADVANTAGES DISADVANTAGES
External style sheets are used to control the style of multiple documents from one file. The user must download the CSS file before seeing the page.
The styles of multiple documents can be controlled from one file. The page may not display correctly if the user has a slow internet connection or if the server is down.
We can use External CSS files multiple time. If the CSS file is not cached, it will have to be downloaded every time the page is loaded.
