QUESTION-2

What are the Disadvantages of using CSS?




INTRODUCTION:

-   CSS is a Design Language.
-   By using CSS, we can make our Website more Attractive.
-   CSS stands for cascading style sheets.
-   CSS is Style Sheet Language.

DISADVANTAGES:

-  Be Ready to Fix HTML Tags: In practice, It is necessary to fix not only one CSS file but also HTML tags associated with CSS selectors.
-   It sometimes increases the editing time and testing as well.
-  Lack of security: Anyone who has a read/write access to a website can change the CSS file.
-  Browser Issue: After designing a webpage it is not sure that the page is going to work similarly in every browser thus it can show different results in different browsers.
-  Can be Confusing: The different levels of CSS like CSS1, CSS2, and CSS3 are confusing for many beginners but they can be understood easily after.

CONCLUSION:

-   There are many Advantages of CSS, but Disadvantages too.
