QUESTION-6

How can the background color of an element be changed?




-   Background-color property sets the background-color of an element.
-   If we apply Background-color property to the specific element, it covers the total size of an element including padding.

SYNTAX:

body{
    background-color: rgb(0, 104, 99);
}

Where,
  body - Selector
  background-color - Attribute
  rgb(0, 71, 67) - value allocated to the Attribute

-   If you want to change Background-color of the element, there are few ways to apply CSS:

1. Inline CSS:

<body style="border: 2px solid black; padding: 5px; width: auto; background-color: #006863; color: white; border-radius: 10px;">

  //content

</body>


2. Internal CSS:

<style>
  body{
    background-color: #006863;
  }
</style>


3. External CSS:

body{
    background-color: #006863;
}


-   Few ways to apply value to the Back-ground Attribute:

1. HEX Value:

body{
    background-color: #006863;
}


2. rgb Value:

body{
    background-color: rgb(0, 104, 99);
}


3. rgba Value:

body{
    background-color: rgba(0, 104, 99, 0.5);
}


4. hsl Value:

body{
    background-color: hsl(177, 100%, 20%);
}


5. hsla Value:

body{
    background-color: hsla(0, 104, 99, 0.5);
}
