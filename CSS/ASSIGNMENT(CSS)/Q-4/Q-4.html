QUESTION-4

Name a few CSS style components.




CSS STYLE COMPONENTS:

-   If you want to style an HTML element, you have to declare the HTML element and give it the style you want.

-   There are few example of CSS Style Components:
    1.SELECTORS
    2.PROPERTIES
    3.VALUES

1.SELECTORS : HTML element name, id name, class name

2.PROPERTIES : Attributes

3.VALUES : Values allocated to the Attributes

SYNTAX:

header{
    font-family: sans-serif;
}

Where,
  header - Selector
  font-family - CSS Style
  sans-serif - CSS Style Value
