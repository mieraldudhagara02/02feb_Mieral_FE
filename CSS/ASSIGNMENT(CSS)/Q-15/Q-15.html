QUESTION-15

What are the external style sheets?




-   There are three ways to apply CSS.
  1. INLINE CSS
  2. INTERNAL CSS
  3. EXTERNAL CSS
1. INLINE CSS: <p style="color: pink;">
2. INTERNAL CSS: <style>    p{color: pink;}    </style>
3. EXTERNAL CSS: p{color: pink;}

-   In "external css", we have to Create separate .css file and then attach this file in our HTML document through HTML <link> tag.

EXAMPLE:

EXTERNAL CASCADING STYLE SHEET

CODE:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>external CSS</title>
<link rel="stylesheet" href="Assignment[CSS].css">
</head>
<body>
<div class="external">EXTERNAL CASCADING STYLE SHEET</div>
</body>
</html>
</div>
