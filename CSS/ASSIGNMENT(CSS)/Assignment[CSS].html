Q-3.     What is the difference between CSS2 and CSS3?



CSS2 VS. CSS3
Sr. CSS2 CSS3
1. CSS2 was released in 1998. CSS3 was released in 1999.
2. In CSS2, there are more styling option than CSS. In CSS3, there are many more styling option along with the support for animation.
3. With CSS2 only 'Web Safe Fonts' can be used. With CSS3 special fonts can be used such as those in 'Google Fonts' and 'TypeCast'.
4. CSS2 Doesn't support the Border-Box Property. CSS3 supports the Border-Box Property.
5. CSS2, Doesn't support the round border property. CSS3, supports the property of round border.
6. CSS2, Doesn't support Grid System and Template Layout. CSS3, supports the Grid System and Template Layout.

Q-5.     What do you understand by CSS Opacity?



-   Opacity means Transparency.
-   The Opacity Property sets the Transparency level for an element.
-   Where 1 is not Transparent.
-   0.5 is 50% Transparent.
-   0 is completely Transparent.

SYNTAX:

    filter: opacity:(value);
}


EXAMPLE:

OPACITY-0

Q-9.     Which property controls the image scroll in the background?



-   If you don't want to scroll your background image, then you have to use the background-attachment property.
-   Or you want your background image is fixed, then also you need to use the background-attachment property.
-   The background-attachment property sets whether a backgroung image scrolls with the page or is fixed.

SYNTAX:

body{
   background-attachment: value;
}


EXAMPLE:

When we apply 'scroll' value in the Background-attachment property,
the background image will scroll with the page.

Specificity:
  1. Inline CSS
  2. ID selectors
  3. Class selectors
  4. Elements
How to Calculate Specificity?
- Start at 0, add 100 for each ID value.
- Add 10 for each class value (or pseudo-class or attribute selector).
- Add 1 for each element selector or pseudo-element.
- Inline style gets a specificity value of 1000, and is always given the highest priority!


Q-13.     What are the ways to integrate CSS as a web page?



-   There are three ways to integrate CSS as a web page.
  • INLINE CSS
  • INTERNAL CSS
  • EXTERNAL CSS
  • INLINE CSS:
 -   In Inline CSS, You can add CSS styles directly to an HTML element using the "style" attribute.
    <p style="color: pink;">

INLINE CASCADING STYLE SHEET

</html>
</div>

Q-15.     What are the external style sheets?



-   There are three ways to apply CSS.
  1. INLINE CSS
  2. INTERNAL CSS
  3. EXTERNAL CSS
1. INLINE CSS: <p style="color: pink;">
2. INTERNAL CSS: <style>    p{color: pink;}    </style>
3. EXTERNAL CSS: p{color: pink;}

-   In "external css", we have to Create separate .css file and then attach this file in our HTML document through HTML <link> tag.

EXAMPLE:

EXTERNAL CASCADING STYLE SHEET

CODE:

Q-17.     What is the meaning of CSS Selector?



-   CSS Selector is used to select the HTML elements you want to style.
-   There are a wide variety of CSS selectors available.
-   If you have more than one element and you want to apply same CSS on them, you need to use selector.
EXAMPLE:

HEADING-1

HEADING-2

-   I want to apply same CSS property to Heading Tags, so I have to use selector.

SYNTAX:

.selector h1, h2{
   background-color: rgba(2, 77, 73, 0.5);
   color: white;
}


-   So, here are some Examples of Selectors:

TYPES OF SELECTORS:
Selector:
h1{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, "h1" is selector.

Class Selector:
.class{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, ".class" is Class Selector.

ID Selector:
#id{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, "#id" is ID Selector.

Pseudo-Classes:
a:hover{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, "a" is Selector and ":hover" is Pseudo-Class Selector.

Pseudo-Element:
p::first-line{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, "p" is Selector and "::first-line" is Pseudo-Element Selector.

Universal Selector:
*{
    background-color: rgba(2, 77, 73, 0.5);
}
-   Here, "*" is Universal Selector.

-   Thus, When we want to apply same CSS to one or more Element we have to use Selectors.


Q-18.     What are the media types allowed by CSS?



-   CSS supports different media types for different devices.

-   The media types that are allowed by CSS are:
  • all
  • print
  • screen
  • speech
  1. all : The all media type is used to apply styles to all devices.
  2. print : The print media type is used for printers.
  3. screen : The screen media type is used for computer screens, tablets, smart-phones, etc.
  4. speech : The speech media type is used for screen readers.
-   Thus, these all are the media types, which is allowed by CSS.


Q-19.     What is the rule set?



-   A CSS rule set consists of a selector and a declaration block.
  [The CSS declaration block is the lines between the curly braces.]
-   The selector points to the HTML element you want to style.
-   The declaration block contains bunch of properties separated by semicolons.

-   ".container" indicates the name of the Class Selector.
-   "first-child" indicates the pseudo-class.
-   Elements inside curly brackets is declaration block which contains some CSS properties and their values.
