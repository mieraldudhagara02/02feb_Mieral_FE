QUESTION-7

How can image repetation of the background be controlled?




-   If you want to set background-image to the content, you have to use background-img Attribute.
-   When you apply background-img property to the element, it's repeat background-image by default vertically and horizontally.

SYNTAX:

body{
   background-repeat: value;
}


EXAMPLE:

1. BACKGROUND REPEAT

When we apply Background-repeat property, background-image repeated both Horizontally and Vertically.
.repeat{
    height: 300px;
    width: 350px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 50%;
    background-repeat: repeat;
    margin: 10px;
    float: left;
}
2. BACKGROUND NO-REPEAT

When we apply Background-no-repeat property, background-image not repeated.
.repeat{
    height: 300px;
    width: 350px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 50%;
    background-repeat: no-repeat;
    margin: 10px;
    float: left;
}
3. BACKGROUND REPEAT-X

When we apply Background-repeat-x property, background-image repeated only Horizontally.
.repeat{
    height: 300px;
    width: 350px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 50%;
    background-repeat: repeat-x;
    margin: 10px;
    float: left;
}
4. BACKGROUND REPEAT-Y

When we apply Background-repeat-y property, background-image repeated only Vertically.
.repeat{
    height: 300px;
    width: 350px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 50%;
    background-repeat: repeat-y;
    margin: 10px;
    float: left;
}
