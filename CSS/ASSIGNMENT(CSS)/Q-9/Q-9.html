QUESTION-9

Which property controls the image scroll in the background?




-   If you don't want to scroll your background image, then you have to use the background-attachment property.
-   Or you want your background image is fixed, then also you need to use the background-attachment property.
-   The background-attachment property sets whether a backgroung image scrolls with the page or is fixed.

SYNTAX:

body{
   background-attachment: value;
}


EXAMPLE:

1. SCROLL

When we apply 'scroll' value in the Background-attachment property,
the background image will scroll with the page.
This is default.
.scroll{
    height: 300px;
    width: 450px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 60%;
    background-repeat: no-repeat;
    background-color: rgb(0, 111, 106);
    margin: 10px;
    float: left;
    background-position: center;
    background-attachment: scroll;
}
2. FIXED

When we apply 'fixed' value in the Background-attachment property,
the background image will not scroll with the page.
.fix{
    height: 300px;
    width: 450px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 60%;
    background-repeat: no-repeat;
    background-color: rgb(0, 111, 106);
    margin: 10px;
    float: left;
    background-attachment: fixed;
}
3. LOCAL

When we apply 'local' value in the Background-attachment property,
the background image will scroll with the element's contents.
.local{
    height: 300px;
    width: 450px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 60%;
    background-repeat: no-repeat;
    background-color: rgb(0, 111, 106);
    margin: 10px;
    float: left;
    background-position: center;
    background-attachment: local;
}
