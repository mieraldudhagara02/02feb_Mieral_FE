QUESTION-13

What are the ways to integrate CSS as a web page?




-   There are three ways to integrate CSS as a web page.
  • INLINE CSS
  • INTERNAL CSS
  • EXTERNAL CSS
  • INLINE CSS:
 -   In Inline CSS, You can add CSS styles directly to an HTML element using the "style" attribute.
    <p style="color: pink;">
EXAMPLE:

INLINE CASCADING STYLE SHEET

CODE:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>inline CSS</title>
</head>
<body>
<div class="inline" style="height: 300px; width: 300px; border: dashed 2px black; background-color: rgb(0, 111, 106); color: white; text-align: center; line-height: 18; margin: auto;">INLINE CASCADING STYLE SHEET</div>
</body>
</html>
</div>

  • INTERNAL CSS:
 -   In Internal CSS, You can add CSS styles to the head section of an HTML document using the "style" tag.
    <style>    p{color: pink;}    </style>
EXAMPLE:

INTERNAL CASCADING STYLE SHEET

CODE:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>inline CSS</title>
<style>
 .internalCSS{
    height: 300px;
    border: dashed 2px black;
    background-color: rgb(0, 111, 106);
    color: white;
    text-align: center;
    line-height: 18;
    margin: auto;
    width: 300px;
}
</style>
</head>
<body>
<div class="internalCSS">INLINE CASCADING STYLE SHEET</div>
</body>
</html>
</div>

  • EXTERNAL CSS:
 -   In External CSS, You can create a separate CSS file and link it to your HTML document using the "link" tag in the head section.
    p{color: pink;}
EXAMPLE:

EXTERNAL CASCADING STYLE SHEET

CODE:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>external CSS</title>
<link rel="stylesheet" href="Assignment[CSS].css">
</head>
<body>
<div class="External">EXTERNAL CASCADING STYLE SHEET</div>
</body>
</html>
</div>
