QUESTION-10

Why should backgound and color be used as separate properties?




BACKGROUND:


-   The background property is a 'shorthand' property.

EXAMPLE:


CSS Code:
.background{
    height: 400px;
    width: auto;
    background: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg) rgb(0, 111, 106) fixed no-repeat center;
    background-size: 60%;
}

In background property,

    url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg) : This is the path of background image.

    rgb(0, 111, 106) : Background Color.

    fixed : Background Attachment.

    no-repeat : The value, we allocated to background-repeat property.

    center : The value, we allocated to background-position property.


COLOR:
