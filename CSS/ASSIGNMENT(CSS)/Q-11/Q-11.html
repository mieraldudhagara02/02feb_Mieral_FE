QUESTION-11

How to center block elements using CSS1?




-   The elements taking full space is known as 'Block Element'.

EXAMPLE:

THIS IS PARAGRAPH TAG.

THIS IS ANOTHER PARAGRAPH TAG.

-   Thus, Block elements are force to another element to start on a new line. However, we don't break it.
-   For centering bloack element, we need to use margin property.

EXAMPLE:

BLOCK ELEMENT
CSS Code:
.center{
    height: 300px;
    width: 300px;
    margin: auto;
    background-color: rgb(0, 111, 106);
    color: white;
    border: solid 2px black;
    border-radius: 10px;
    text-align: center;
    line-height: 18;
}
-   By using margin property, we can controlling them horizontally and vertically.
-   We need to specify the margin value from left and right such that it looks centered.
-   We can use 'auto' value, that is automatically sets the block element in center.

EXAMPLE:

WITHOUT MARGIN
MARGIN AUTO
MARGIN TOP
MARGIN RIGHT
MARGIN BOTTOM
MARGIN LEFT

-   'Margin' is shorthand property.
 HOW!!!
-   When we use 'margin:10px', all four margins are 10px.

EXAMPLE:

MARGIN
CSS Code:
.margin{
    width: 50%;
    margin: 50px;
    background-color: white;
    color: rgb(0, 111, 106);
    padding: 10px;
    border: 2px solid black;
    text-align: center;
}

-   top and bottom, left and right margins are 50px.
-   Thus, we have to use margin property for centering block element.

NOTE: If we want to centering 'block element' without using margin property, than we have to use 'display: flex;' and then we can centering block element through 'justify-content'[Horizontally] and 'align-item'[Vertically].
