QUESTION-8

What is the use of the background-position property?




-   The background-position property sets the starting position of a background image.
-   By default, a background-image is placed at the top-left corner of an element, and repeated both vertically and horizontally.

SYNTAX:

body{
   background-position: value;
}


EXAMPLE:

1. TOP-LEFT

When we apply 'top left' value in the Background-position property, background-image will be displayed on
top-left side of the content.
#top-left{
    height: 300px;
    width: 350px;
    background-image: url(https://i.pinimg.com/originals/c2/04/89/c20489420ccb827dff570047e568fbad.jpg);
    background-size: 50%;
    background-repeat: no-repeat;
    background-color: rgb(0, 111, 106);
    margin: 10px;
    float: left;
    background-position: top left;
}
