QUESTION-5

What do you understand by CSS Opacity?




-   Opacity means Transparency.
-   The Opacity Property sets the Transparency level for an element.
-   Where 1 is not Transparent.
-   0.5 is 50% Transparent.
-   0 is completely Transparent.

SYNTAX:

selector{
    filter: opacity:(value);
}


EXAMPLE:

OPACITY-1
OPACITY-0.5
OPACITY-0

-   Opacity adds Transparency to the background of an element and all of it's child class too.
-   If we write some content inside the element, and we gave Opacity value 0 to the element.
-   With Background, Content also completely transparent. That's why it's hard to read it.
-   So, Solution of this problem is we can use alpha property [RGBA Color Values] instead of Opacity.
