QUESTION-14

What is embedded style sheet?



-   Embedded style sheet is also known as "Internal CSS."
  • INTERNAL CSS:
 -   In Internal CSS, You can add CSS styles to the head section of an HTML document using the "style" tag.
    <style>    p{color: pink;}    </style>
EXAMPLE:

INTERNAL CASCADING STYLE SHEET

CODE:
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta http-equiv="X-UA-Compatible" content="IE=edge">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>inline CSS</title>
<style>
 .InternalCSS{
    height: 300px;
    border: dashed 2px black;
    background-color: rgb(0, 111, 106);
    color: white;
    text-align: center;
    line-height: 18;
    margin: auto;
    width: 300px;
}
</style>
</head>
<body>
<div class="InternalCSS">INLINE CASCADING STYLE SHEET</div>
</body>
</html>
</div>
