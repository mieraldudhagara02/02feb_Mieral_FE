background-color : Specifies the background color to be used [css-1]

background-image : Specifies ONE or MORE background images to be used
background-repeat : Specifies how to repeat the background images

The background-attachment property sets whether a background image scrolls with the rest of the page, or is fixed.

background-size : Specifies the size of the background images

CSS linear-gradient property lets you display smooth transitions between two or more colors.

The radial-gradient() function sets a radial gradient as the background image.
A radial gradient is defined by its center.
